# Chapter1:出发

有一名朋友,最近沉迷星际拓荒，在相关视频下刷到《挽救计划》这部影片，便萌生了一探究竟的想法，顺便带上了我。作为一部小说改编的超长科幻电影:spoiler[长达整整2个半小时],在影院亮灯的一瞬间，我的感受是 "酣畅淋漓"，在我的评价中已经达到佳作水准。这部片子最好的有点如果让我总结的话，就是 "逻辑完整且剧情正常"。

客观来说《挽救计划》的制作水准绝对是出色的，虽然没有很多《流浪地球》中注重"大场面"来进行宏大叙事的视觉冲击，但是其将很多细节打磨的非常好，以一种 "纪实叙事" 将信息反馈给幕前的观众，而这一特性在与外星人从接触到熟悉这一段中将会体现的更加充分。

# Chapter2:第一次接触

"第一次接触" 是一个专有名词，表示人类对地外智慧生命的第一次接触与交流。在有 "外星人" 这样题材的科幻中，将 "外星人" 与 "人类" 的关系塑造清楚且逻辑正常其实是一个非常困难的事情。很容易陷入一种 "人类中心主义" 的陷阱之中，而这个片子巧妙的将两个文明的第一次接触从两个个体的接触展开。

这其实是一个高明的写法,描述两个文明中"文明的冲突"比两个个体的"个体的冲突"会难写的多，因为在个人中可以放大个体所展现的 "非理性" 的部分:spoiler[爱与和平有什么不好的!]将猜疑与揣测转化为一种纯粹的处于个体内心的"冲动感",文明会去需要考虑牺牲自己是不是值得的但是个体并不会，在这部片子中主角和洛基直接先后的救赎就是这样的体现。空旷的太空与孤身一人:spoiler[(一石头?)]的孤独感让两个个体成为互相支撑他人的存在，而这种属于个体的强烈情感不仅可以让导演更好的发挥出属于 "文明的温暖面" 的叙事，也能让我们更好体会这部片子的主旨："在面对未知的文明时我们是否有勇气拯救对面"

# Chapter3:红色的飘带

评价科幻作品，我们通常观察其三个内核:逻辑,想象力,深层人文属性。就像"喜剧的内核是悲剧一般",科幻作品最终讨论的 "在面对一个可能存在的情形中，人(人类)会怎么做"。《星际穿越》让我们体会到了人类在面对绝对未知时候展现的勇气，《流浪地球》告诉我们人类团结的力量能有多大。这部剧则从人的视角出发，传达了"文明的温暖面"这样的主旨。

逻辑叙事方面，本作的插叙做的非常优秀。:spoiler[我一开始只是以为是纯粹的插叙，后面仔细一想才发现这其实是主角被"绑"到飞船后逐渐找回记忆的过程]同时，在科学方面的严谨性做的也不错,将噬星体从被发现到捕获利用，到主角在天仓五中收集捕食者等描述，逻辑是合理的且具有想象力的。

# Chapter4:星河中的文明

其实仔细一想，这篇文章特意提及噬星体已经感染了几乎所有文明，这就和人类共同面对一个困难一样。全宇宙的文明都共同面对一个问题。所以到这样的一个层面，洛基说的那句 "拯救地球，拯救波江星"其实就是两个人的抉择，而这样的抉择从两人:spoiler[外星人也是人(]互相选择为对方牺牲的决定而体现。这正是一种善良与温暖，而就是这样的善良与温暖同时救了地球和波江星。

环境塑造方面，这部片子可谓是致敬了一堆片，而且其对于太空遥远距离的 "孤独感" 的氛围塑造也是不得不品的，而真是这种出色的孤独塑造，让两个在太空流浪的文明个体有了交流和依靠。

总的来说，这是一部质量上乘的，故事逻辑清晰完整且能让我一下子全神贯注投入到其故事中的优秀科幻作品，好评喵
