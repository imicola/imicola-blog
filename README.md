# imicola-blog

个人博客站点，基于 [Astro](https://astro.build/) + [Svelte](https://svelte.dev/) + [Tailwind CSS](https://tailwindcss.com/) 构建，使用 [Mizuki](https://github.com/Katherine1918/mizuki) 主题。

## 站点

- 地址：[imicola.com](https://imicola.com/)

## 技术栈

- **框架**：Astro 5（静态站点生成）
- **UI**：Svelte 5 + Tailwind CSS 4
- **部署**：Vercel
- **包管理**：pnpm
- **搜索**：Pagefind
- **评论**：Twikoo
- **数学公式**：KaTeX（remark-math + rehype-katex）
- **图表**：Mermaid
- **代码高亮**：Expressive Code

## 主要功能

- 博客文章（支持 Markdown / MDX）
- 相册 / 图片展示
- 追番记录
- 友链页面
- 日记
- 项目展示
- RSS / Atom 订阅
- 全站搜索（Pagefind）
- 页面过渡动画（Swup）
- 樱花特效

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build

# 预览构建结果
pnpm preview
```
